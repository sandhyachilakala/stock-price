# 📈 Stock Market Movement Prediction Using Machine Learning

## Overview

This project predicts stock market movements (Upward or Downward) using Machine Learning techniques and stock market data. Multiple classification algorithms were implemented, evaluated, and compared to identify the most accurate model for trend prediction.

## Features

* Data cleaning and preprocessing
* Feature engineering and scaling
* Exploratory Data Analysis (EDA)
* Model training and comparison
* Performance evaluation using Accuracy, Precision, Recall, and F1-Score
* Feature importance analysis

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* XGBoost

## Machine Learning Models

* Logistic Regression
* Decision Tree
* K-Nearest Neighbors (KNN)
* Support Vector Machine (SVM)
* Naive Bayes
* XGBoost
* Random Forest

## Results

The Feature-Engineered Random Forest model achieved the best performance among all models.

**🏆 Final Accuracy: 92.15%**

### Model Comparison

| Model                            | Accuracy   |
| -------------------------------- | ---------- |
| Logistic Regression              | 64.08%     |
| Decision Tree                    | 73.74%     |
| SVM                              | 72.35%     |
| KNN                              | 71.63%     |
| Naive Bayes                      | 64.08%     |
| XGBoost                          | 83.17%     |
| Random Forest                    | 89.59%     |
| Feature-Engineered Random Forest | **92.15%** |

## Key Insights

* Feature engineering significantly improved prediction performance.
* Random Forest outperformed all other classification models.
* Trading volume and price-related indicators were among the most important features influencing predictions.
